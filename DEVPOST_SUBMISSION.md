# Devpost Submission Content

## Project Title

MyManfaat Agent: AI Benefit Navigator for Malaysians

## Tagline

Making public support easier to find, understand, and apply for.

## Inspiration

I was inspired by the challenge many people face when trying to find support programmes. Aid, grants, and training opportunities may exist, but people often do not know what they qualify for or how to apply. I wanted to create a practical AI agent that guides users from confusion to action.

## What It Does

MyManfaat Agent interviews users, understands their profile, searches a benefit programme database, ranks suitable programmes, explains eligibility in simple language, and generates a document checklist plus application action plan.

## How We Built It

The prototype was built using React and Vite. A sample benefit database was created with programme categories, eligibility rules, document requirements, and application steps. The matching engine scores each programme based on age, income category, user profile, and selected need.

The planned full version will use Gemini for AI explanation, Google Cloud Agent Builder for workflow, and MongoDB to store programme records and eligibility data.

## Challenges We Ran Into

The main challenge was keeping the project realistic for a hackathon while still showing a meaningful AI agent workflow. Public benefits can be complex, so the prototype uses a focused sample dataset and clear scoring logic to demonstrate the concept.

## Accomplishments

I created a working prototype that demonstrates a complete user journey from eligibility questions to personalised recommendations, document checklist, and application steps. The project also has a clear social impact direction and can be expanded with real verified programme data.

## What We Learned

I learned how to break down a social-impact problem into a practical AI agent workflow. I also learned that a useful AI product should not only answer questions, but guide users through decisions and next steps.

## What's Next

Next, MyManfaat Agent can be improved by connecting to verified official programme sources, adding Gemini-powered explanations, integrating MongoDB for dynamic benefit records, supporting Bahasa Malaysia, and allowing users to save deadlines or receive reminders.
