# Demo Video Script — 3 Minutes

## 0:00–0:20 — Problem
Many Malaysians miss out on aid, grants, training programmes, and support schemes because the information is scattered and difficult to understand.

## 0:20–0:40 — Solution
This is MyManfaat Agent, an AI-powered benefit navigator that helps users discover support programmes they may be eligible for.

## 0:40–1:40 — Live Demo
First, the user answers simple questions such as age, state, employment status, income category, and the type of help they need.

For this demo, I will use a 50-year-old Malaysian employee looking for training and reskilling support.

After clicking Run Agent, the system searches the benefit database and ranks the most suitable programmes.

## 1:40–2:20 — Recommendation Output
The agent recommends Reskilling for Mid-Career Workers as the best match. It explains why the programme fits the user profile, shows the match score, and lists other suitable options.

It then creates a document checklist and a step-by-step application action plan.

## 2:20–2:45 — Technology
The current prototype is built with React, Vite, and a sample benefit database. The planned full version uses Gemini for reasoning and explanation, Google Cloud Agent Builder for workflow, and MongoDB to store benefit and eligibility records.

## 2:45–3:00 — Closing
MyManfaat Agent makes public support easier to find, easier to understand, and easier to apply for.
